## 说明
```
可先下载demo，并更新demo项目依赖路径到 sdk/0.0.X，然后运行demo

在项目中添加sdk目录中最新的dll包：
   |-- mimc-csharp-sdk.dll
      |--Newtonsoft.Json.dll
      |--protobuf-net.dll
      |--StreamJsonRpc.dll
      |--log4net.dll
```
